# Triple Espresso

# Descripción

El proyecto es una página web que permite darle la bienvenida a los clientes a la cafetería Triple Espresso, permite que los clientes hagan una reservación en la tienda y que puedan conseguir más información.

# Funcionalidad

El header de la página web está diseñado por medio de flexbox para colocar le icono y la lista de navegación, hay un elemento de título con un parrafo que tiene la debida separación entre la primera flexbox y la segunda, hay una segunda flexbox que contiene la infomración adicional de la cafeteria; estos elementos se ajustaron principalmente con margin y position absolute. Como último también se aplicó fondo y una imágen del cafe con position absolute.

El main contiene un formulario para hacer una reservación en el café con la ayuda de los elementos tipos form y las subclases que se le pueden asignar.

El footer contiene la información adicional así como una despedida del sitio web.

# oportunidades o planes de mejora a futuro

El sitio web puede tener mejor interacción con las personas al vincular google maps para que los clientes potenciales puedan ver la ubicación del café viendo una imagen o que al seleccionarla te lleve a la aplicación de google maps.
